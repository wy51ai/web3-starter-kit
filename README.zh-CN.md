# web3-starter-kit

[English](README.md)

一个面向Web3新手友好型Skills套件。基于 [okx/onchainos-skills](https://github.com/okx/onchainos-skills) 提供的链上数据能力，封装成一套以安全为先、带引导式交互的链上交易体验，适合刚接触 Web3 的用户。

## 功能概览

| 功能 | 技能 | 说明 |
|---|---|---|
| **安全评分** | web3-safe-guide | 从 5 个维度为任意代币打出 0–100 分：社区认证、流动性、持币地址数、前 10 持仓集中度，以及蜜罐检测 |
| **聪明钱信号** | web3-safe-guide | 跟踪 Smart Money、Whales、KOL 最近在买什么，并结合信号触发后的 K 线走势验证动能 |
| **热门代币发现** | web3-safe-guide | 展示当天最热的代币，并先做安全过滤，明显骗局不会出现在结果里 |
| **安全换币引导** | web3-safe-guide | 提供分步式 swap 流程，包含强制安全检查、预估 gas，以及广播后的交易跟踪 |
| **模拟交易器** | web3-trade-simulator | 用虚拟的 10,000 美元练习交易；价格数据来自真实链上行情，不涉及真实资金，并附带 5 个维度的决策评分 |

## 安全评分体系

每次代币分析都会先自动给出一个安全评分：

```text
90–100  🟢 安全       — 相对低风险
70–89   🟡 谨慎       — 存在一定风险，建议控制仓位
50–69   🟠 警告       — 风险较高，仅适合小额尝试
0–49    🔴 危险       — 风险很高，不建议参与
Honeypot ☠️ 骗局      — 买入后可能无法卖出，系统会直接拒绝
```

## 模拟交易评分体系

每笔模拟买入都会从 5 个维度给出入场决策评分（满分 100）：

| 维度 | 权重 | 评分依据 |
|---|---|---|
| 安全评分 | 30 分 | 将代币安全分（0–100）线性映射到 0–30 分 |
| 聪明钱对齐 | 20 分 | 有信号且持有中 +20；信号已消退 +0；已完全出逃 −10 |
| 趋势时机 | 20 分 | 顺势买入 +20；横盘 +10；逆势买入 +0 |
| 流动性纪律 | 15 分 | >$1M +15；$100K–$1M +10；$10K–$100K +5；<$10K −15 |
| 仓位管理 | 15 分 | ≤10% 资金 +15；11–25% +10；26–50% +5；>50% −15 |

```text
85–100  🏆 专家       — 纪律严明、研究充分的入场
65–84   🟢 良好       — 决策扎实，有小瑕疵
45–64   🟡 一般       — 有一定直觉，需在薄弱维度加强
25–44   🟠 冒险       — 忽视了重要风险信号
0–24    🔴 危险       — 这种交易模式在真实市场中大概率亏损
```

## 安装

**第一步：安装依赖 onchainos-skills**（提供所有链上数据能力）

```bash
npx skills add okx/onchainos-skills
```

**第二步：安装本套件**

```bash
npx skills add wy51ai/web3-starter-kit
```

支持 Claude Code、Cursor、Codex CLI、OpenCode 等工具，自动检测当前环境并按需安装。

## 工作原理

这个技能套件包含 2 个技能，共同串联成一套协同工作流：

### web3-safe-guide

| 子技能 | 用途 |
|---|---|
| `okx-dex-token` | 代币搜索、价格信息、持仓分布、热门榜单 |
| `okx-dex-market` | 实时价格、K 线数据、聪明钱信号 |
| `okx-dex-swap` | Swap 报价与交易数据 |
| `okx-onchain-gateway` | Swap 前 gas 预估，以及交易状态跟踪 |

在这些原始数据之上，额外提供了：安全评分、基于信号后 K 线的动能分析、面向新手的通俗解释、swap 前的安全闸门，以及引导式对话流程。

### web3-trade-simulator

| 子技能 | 用途 |
|---|---|
| `okx-dex-token` | 代币搜索、价格信息（获取模拟入场 / 出场价） |
| `okx-dex-market` | K 线趋势判断、聪明钱信号对齐 |
| `okx-dex-swap` | 模拟买卖报价（只读，**不会**发起真实交易） |

模拟器在真实链上数据之上额外提供了：虚拟持仓账本、5 维入场评分、出场质量加分、整体交易能力报告，以及内嵌的教学时机提示。虚拟仓位状态持久化保存在本地文件 `~/.web3-trade-simulator/portfolio.json`，跨对话保留，重新开始对话可接续上次的仓位。
